\## 关键数学（见方案文档第二章）



\### wrap\_detect

判断 x+r 是否 >= 2^64（溢出）

返回 w=1 表示溢出，w=0 表示未溢出

参考 MCU 论文 Protocol 4，以及引用的 Bicoptor 论文



\### Π\_exp（4轮）

核心恒等式：e^x = e^(x+r) · e^(w·2^64 - r)

其中 w = wrap(x, r)



\### Π\_softmax（6轮）

核心技巧：引入随机掩码 t 隐藏分母

见方案文档 2.5 节



\### Π\_gelu（8轮）

GeLU(x) = x · sigmoid(1.702x)

先实现 sigmoid（6轮），再调用 Π\_mul（2轮）
